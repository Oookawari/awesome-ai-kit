# 贡献说明

这个仓库公开可读，但主要由个人维护。欢迎通过 Issue 或 Pull Request 提供修正和有明确复用价值的内容。

提交内容时请遵守以下约定：

- 按 Prompt、Skill、Workflow、AI Coding 技巧或模板分类。
- 每项内容使用独立的英文连字符目录名。
- 主要说明使用中文，技术名称和代码可保留英文。
- 不得包含个人信息、密钥、私人对话、敏感项目数据或未公开研究内容。
- 自有内容可以完整提交；第三方内容只提供简短说明、适用场景和原始链接。
- 无法确认原始来源的第三方内容不予收录。
- 添加或调整条目时，同步更新分类 README 和根 README。

提交即表示你有权提供相关内容，并同意仓库按照 [MIT License](LICENSE) 分发你的原创贡献。第三方链接内容不因此改变其原有许可。
