# Prompt

Prompt 是可以在新对话中单次提交、独立完成任务的指令。依赖此前对话或需要多轮推进的内容应放入 [Workflow](../workflows/README.md)。

## 内容列表

- [提示词优化助手](prompt-optimizer/README.md)：将模糊、碎片化的描述整理成目标明确、可直接使用的 Prompt
- [每日科研简报 Automation 创建 Prompt](daily-research-brief/README.md)：创建定时科研情报任务，每天筛选少量高价值论文、报告、博客与社区动态

## 条目格式

每个 Prompt 使用独立目录：

```text
prompts/<prompt-name>/README.md
```

条目正文依次包含名称、一句话用途、可直接复制的 Prompt、必要的使用说明和更新记录。
